# 🏦 Banking Analytics Project

An end-to-end Banking Analytics project developed using **Excel, SQL, Python, and Power BI**. This project demonstrates the complete data analytics workflow—from data cleaning and preprocessing to exploratory data analysis, data modeling, and interactive dashboard creation. It provides valuable insights into customer behavior, loan performance, credit card usage, operational risk, and customer feedback to support data-driven business decisions.

---

# 📌 Project Overview

The objective of this project is to analyze banking data and transform raw information into meaningful business insights. The project covers the complete analytics lifecycle, including data preparation, SQL querying, exploratory data analysis (EDA), and dashboard development.

---

# 🎯 Business Objectives

- Analyze customer demographics and account behavior.
- Monitor loan performance and approval trends.
- Identify high-risk customers.
- Track credit card utilization.
- Analyze customer feedback and resolution performance.
- Build interactive dashboards for business decision-making.

# 📂 Project Workflow

### 1️⃣ Data Collection
Collected banking data from multiple sources.

### 2️⃣ Data Cleaning (Excel)
- Removed duplicate records
- Handled missing values
- Standardized data
- Corrected data types

### 3️⃣ SQL Analysis
- Data extraction
- Filtering
- Aggregation
- Joins
- Business queries

### 4️⃣ Python Analysis
- Exploratory Data Analysis (EDA)
- Data visualization
- Customer insights
- Trend analysis

### 5️⃣ Power Query
- Data transformation
- Data modeling
- Custom columns
- Conditional columns

### 6️⃣ DAX Measures
Created dynamic KPIs including:
- Total Customers
- Total Deposits
- Total Loans
- Average Account Balance
- Average Loan Amount
- Risk Ratio
- Credit Utilization
- Resolution Rate
- Average Resolution Time

### 7️⃣ Dashboard Development
Designed interactive dashboards with filters, slicers, KPIs, and business visualizations.
